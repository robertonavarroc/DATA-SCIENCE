- Codigos utilizados para trabajar con python
- :eye: Cuando instales python siempre utlizar una version estable, no la ultima, adicional siempre marcar la opcion **`Path`**.

```
python --version
``` 
- Para ver la version de python, ir a cmd y colocar el codigo **`"python --version"`**.

```
python
```

```

```
